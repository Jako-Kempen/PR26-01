Project description:
The project relates to the design of a smart sensor, a device with WiFi and Bluetooth cababilities that communicates sampled signals to a private local webserver.

File Hierarcy:
-> Bill of Material 			(Contains the breakdown of each element/module of the project/product)
-> Design Reports			(Contains the specification document folder and the Design report and the final specification document)
	-> Specification Documents 	(Contains all versions of the specification document)
-> Electrical Documents			(Contains all electrical documents [Altium files, PCB designs, Wiring, ect.] and Spice simulation documents)
	-> Obsolete Files		(Contains all previous versions of electrical documents sorted folders by version)
	-> Source Files			(Contains latest version of documents and Altium project)
	-> Spice Simulations		(Contains all SPICE Simulations used during the design process)
	*Change Log indicating changes between versions
-> Firmware				(Contains all firmware files, final firmware design doument and the obsolete files of prevoius versions)
	-> Documents			(Contains all previous firmware design documents, used as reference in the Design Documents)
	-> Obsolete Files		(Contains all previous versions of the firmware)
	-> Sources			(Contains latest version of the firmware used)
	*Change log for each files	(Indicates the change between versions of each file with a timestamp)
-> Mechanical Documents			(Contains all versions of all the parts + All assemblies, Dimentional drawings of each as well as STL files if used)
-> Software				(Contains all versions of software source code)
	-> Obsolete Files		(Contains all previous versions of source code and executables)
	-> Source			(Contains the latest version of the source code)
	*Latest Executable		(Latest executable of the software)
	*Change log for each file	(Indicates the change between versions of each file with a timestamp)
